# Interactive EMR Simulation Templates
**Developer:** Stephani Hamilton, Ed.D.
**Status:** Proprietary Educational Resource (2026)

## Overview
This repository contains interactive HTML/JS EMR simulations designed for clinical education. 
These files are used to simulate real-world medical record entry in a controlled classroom environment.
